
# ethProjectv3



## Run Locally

Clone the project

```bash
  git clone https://github.com/ksheetiz/ethprojectv3.git
```

Go to the project directory

```bash
  cd ethprojectv3
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn react-native start
```


## Tech Stack

**Client:** React, React-Native, Ethers, Javascript

**Server:** Infura
